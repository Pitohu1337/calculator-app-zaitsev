# Мобильное приложение "Калькулятор"

## О проекте
Курсовая работа по разработке мобильного приложения "Калькулятор" с расширенными функциями

## Функционал
- Базовые математические операции (+, -, ×, ÷)
- Расширенные функции (%, √, x², 1/x)
- История вычислений
- Темная/светлая тема
- Интуитивно понятный интерфейс

## Технологии
- Flutter 3.0+
- Dart
- Android Studio / VS Code

## Установка и запуск
1. Установите Flutter SDK
2. Клонируйте репозиторий
3. Запустите `flutter pub get`
4. Запустите `flutter run`
