# Инструкция по установке и запуску приложения "Калькулятор"

## Автор: Зайцев А.В. Группа 4ИСП9-40

## Требования к системе
- Flutter SDK 3.0 или выше
- Android Studio или VS Code
- Android эмулятор или физическое устройство

## Установка Flutter
1. Скачайте Flutter SDK с официального сайта
2. Распакуйте архив в папку `C:\flutter\`
3. Добавьте путь в переменную среды PATH
4. Проверьте установку: `flutter doctor`

## Запуск проекта
1. Клонируйте репозиторий:
`bash`
git clone https://github.com/Pitohu1337/calculator-app-zaitsev.git

2. Перейдите в папку проекта:
```bash
cd calculator-app-zaitsev

flutter pub get

flutter run

flutter build apk --release
